# Flydar application notes

## Objective
Provide and present winds aloft forecasts in graphical format for given location.  Wind speed, direction, and strngth. 
I'd like to be able to search by location then view what the winds look like (up to 10,000 ft) on an hourly basis
The model i'd like to use for these forcasts are the high rez rapid refresh models.


## features
* Ability to search by zip code or coordinates for hyperlocal weather reports
* Save Favorite locations
* provide additional information such as cloud base / rain / gust strength
